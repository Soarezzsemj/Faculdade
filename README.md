# 📚 Estrutura de Dados (Linguagem C)

Este repositório é dedicado aos exercícios, implementações e estudos desenvolvidos para a disciplina de **Estrutura de Dados**, cursada durante a graduação.

O objetivo é registrar a evolução no aprendizado das principais **estruturas de dados e algoritmos**, além de organizar os materiais produzidos ao longo do semestre.

# 📂 Organização do Repositório

Os códigos estão organizados por **temas estudados ao longo da disciplina**, agrupando exercícios e implementações relacionados a cada conteúdo.

| Tema | Conteúdo Principal |
|-----|--------------------|
| Manipulação de Arquivos | Leitura, escrita e manipulação de dados em arquivos |
| Listas | Implementações de listas e suas variações |
| Pilhas | Estruturas de dados do tipo pilha (LIFO) |
| Filas | Estruturas de dados do tipo fila (FIFO) |
| Tabelas Hash | Implementação e utilização de tabelas hash |
| Algoritmos de Ordenação | Métodos clássicos de ordenação |
| Algoritmos de Busca | Técnicas de pesquisa em estruturas de dados |
